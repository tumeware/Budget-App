# BudgetPal - Budget app

### What is it
A budget calculator app made with React Native (Expo).

### Platforms
- iOS
- Android

### What's included
- Designed from scratch
- Redux
- i18n (multi language)
- Darkmode support
- Typescript
- Admob
- Firebase support
